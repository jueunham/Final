# KHOB
- 구인구직 기능과 기업 상세 정보를 조회할 수 있는 통합 채용 사이트

- 50 만 건 이상 기업 정보 조회

## 사용기술 및 버전
- Java 1.8
- Spring Framework 5.1.8
- Spring Security 5.1.5
- ORACLE XE 11g 
- Apache Tomcat 9.0
- JavaScript, Ajax, Oracle Sql, XML, JSON ...
